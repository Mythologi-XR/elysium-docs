# Projets et sauvegarde cloud

Une fois que vous disposez d'un [compte ELYSIUM](./your-account), vous pouvez enregistrer vos designs dans le cloud et les gérer depuis le panneau **Projets** dans la barre latérale droite.

## Enregistrer dans le cloud

Cliquez sur **"Enregistrer sur ELYSIUM"** dans la zone d'actions de la barre latérale (le bouton avec le logo ELYSIUM). Si vous n'êtes pas encore connecté, l'écran de connexion s'ouvre d'abord.

![Bouton Enregistrer sur ELYSIUM mis en évidence dans la zone d'actions de la barre latérale](/img/card-designer/53.png)

### Le formulaire de sauvegarde

Remplissez les détails pour votre sauvegarde cloud :

| Champ | Requis | À quoi cela sert |
|-------|--------|-----------------|
| **Nom du projet** | Oui | Prérempli avec le nom de votre projet actuel |
| **Nom de la carte** | Oui | Prérempli avec le titre de votre carte |
| **Description** | Oui | Une courte description de votre design |
| **Statut** | Oui | **Brouillon** (en cours) ou **Publié** (prêt à partager) |

Cliquez sur **Enregistrer** pour commencer le téléversement. Une barre de progression affiche le pourcentage et la taille du fichier.

![Modal d'enregistrement sur ELYSIUM avec les champs nom du projet, nom de la carte, description et statut](/img/card-designer/54.png)

![Barre de progression du téléversement affichant le pourcentage complété et la taille du fichier en Ko](/img/card-designer/55.png)

### Après l'enregistrement

Une fois la sauvegarde terminée, vous verrez quatre options :

| Action | Ce qu'elle fait |
|--------|----------------|
| **Continuer à éditer** | Ferme la boîte de dialogue et retourne à votre design |
| **Changer de projet** | Charge une eCarte cloud différente |
| **Partager la carte** | Crée un lien de partage pour la carte que vous venez d'enregistrer |
| **Créer un nouveau projet** | Recommence à zéro avec un assistant de configuration rapide |

![Écran post-sauvegarde affichant les boutons Continuer à éditer, Changer de projet, Partager la carte et Créer un nouveau projet](/img/card-designer/56.png)

## Le panneau Projets

Le panneau **Projets** (icône dossier) dans la barre latérale droite affiche toutes vos eCartes sauvegardées dans le cloud.

### Avant la connexion

Si vous n'êtes pas connecté, le panneau affiche les boutons **Connexion** et **Créer un compte** pour vous aider à démarrer.

![Panneau Projets avant la connexion affichant les boutons Connexion et Créer un compte](/img/card-designer/57.png)

### Votre liste d'eCartes

Après la connexion, vous verrez une liste déroulante de vos eCartes enregistrées. Chaque ligne affiche :

- **Nom du projet** et **nom de la carte**
- **Badge de statut** — Brouillon (jaune), Publié (vert) ou Synchronisé (bleu)
- **Temps** depuis la dernière modification (par ex. "il y a 2h", "il y a 3j")
- **Flèche de développement** — Cliquez pour voir la description et un bouton **Ouvrir**

![Panneau Projets montrant une liste d'eCartes avec des badges de statut, des horodatages et l'eCarte active surlignée en vert](/img/card-designer/58.png)

![Ligne d'eCarte développée montrant le texte de description et le bouton Ouvrir](/img/card-designer/59.png)

L'eCarte sur laquelle vous travaillez est mise en évidence avec une bordure verte et une coche.

### Recherche et navigation

- Utilisez la **barre de recherche** en haut pour filtrer vos eCartes instantanément
- Les boutons **Suivant/Précédent** vous permettent de naviguer dans de grandes collections

![Barre de recherche filtrant les eCartes dans le panneau Projets avec les contrôles de pagination visibles](/img/card-designer/60.png)

### Charger une eCarte cloud

Cliquez sur une eCarte dans la liste pour la charger dans votre espace de travail. Vous verrez une barre de progression de téléchargement avec le pourcentage et la taille du fichier pendant le chargement.

![Barre de progression du téléchargement affichant le pourcentage et la taille du fichier lors du chargement d'une eCarte depuis le cloud](/img/card-designer/61.png)

:::caution
Le chargement d'une eCarte différente remplace votre espace de travail actuel. Si vous avez des modifications non enregistrées, une boîte de dialogue de confirmation vous demande si vous souhaitez rester ou ignorer vos modifications.
:::

![Boîte de dialogue de confirmation "Vous avez des modifications non enregistrées" avec les boutons Rester et Quitter](/img/card-designer/62.png)

## Protection contre les modifications non enregistrées

Le Concepteur de cartes vous protège de plusieurs façons :

- **Changement d'eCarte** — Une boîte de dialogue de confirmation apparaît si vous essayez de charger une eCarte différente avec du travail non enregistré
- **Déconnexion** — Vous serez averti avant de vous déconnecter avec des modifications non enregistrées
- **Fermeture du navigateur** — Une invite native "Quitter le site ?" apparaît si vous essayez de fermer l'onglet avec du travail non enregistré

![Modal de navigation avec avertissement ambré, bouton Rester (violet) et bouton Quitter (gris)](/img/card-designer/63.png)

:::tip
Enregistrez fréquemment en utilisant "Enregistrer sur ELYSIUM" pour garder votre travail en sécurité. Le panneau Projets indique quand chaque eCarte a été enregistrée pour la dernière fois.
:::
