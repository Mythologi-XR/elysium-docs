# Projects and Cloud Save

Once you have an [ELYSIUM account](./your-account), you can save your designs to the cloud and manage them from the **Projects** panel in the right sidebar.

## Saving to the cloud

Click **"Save to ELYSIUM"** in the sidebar actions area (the ELYSIUM logo button). If you're not signed in yet, the sign-in screen opens first.

![Save to ELYSIUM button highlighted in the sidebar actions area](/img/card-designer/53.png)

### The save form

Fill in the details for your cloud save:

| Field | Required | What it's for |
|-------|----------|---------------|
| **Project Name** | Yes | Pre-filled from your current project name |
| **Card Name** | Yes | Pre-filled from your card title |
| **Description** | Yes | A short description of your design |
| **Status** | Yes | **Draft** (work in progress) or **Published** (ready to share) |

Click **Save** to start uploading. A progress bar shows the upload percentage and file size.

![Save to ELYSIUM modal with project name, card name, description, and status fields](/img/card-designer/54.png)

![Upload progress bar showing percentage complete and file size in KB](/img/card-designer/55.png)

### After saving

Once your save completes, you'll see four options:

| Action | What it does |
|--------|-------------|
| **Continue Editing** | Close the dialog and go back to your design |
| **Switch Project** | Load a different cloud eCard |
| **Share Card** | Create a share link for the card you just saved |
| **Create New Project** | Start fresh with a quick setup wizard |

![Post-save screen showing Continue Editing, Switch Project, Share Card, and Create New Project action buttons](/img/card-designer/56.png)

## The Projects panel

The **Projects** panel (folder icon) in the right sidebar shows all your cloud-saved eCards.

### Before signing in

If you're not logged in, the panel shows **Sign In** and **Create Account** buttons to get you started.

![Projects panel before sign-in showing Sign In and Create Account buttons](/img/card-designer/57.png)

### Your eCard list

After signing in, you'll see a scrollable list of your saved eCards. Each row shows:

- **Project name** and **card name**
- **Status badge** — Draft (yellow), Published (green), or Synced (blue)
- **Time** since last edit (e.g., "2h ago", "3d ago")
- **Expand arrow** — Click to see the description and an **Open** button

![Projects panel showing a list of eCards with status badges, timestamps, and the active eCard highlighted in green](/img/card-designer/58.png)

![Expanded eCard row showing the description text and Open button](/img/card-designer/59.png)

The eCard you're currently working on is highlighted with a green border and checkmark.

### Searching and browsing

- Use the **search bar** at the top to filter your eCards instantly
- **Next/Previous** buttons let you navigate through large collections

![Search bar filtering eCards in the Projects panel with pagination controls visible](/img/card-designer/60.png)

### Loading a cloud eCard

Click an eCard in the list to load it into your workspace. You'll see a download progress bar with the percentage and file size as it loads.

![Download progress bar showing percentage and file size while loading an eCard from the cloud](/img/card-designer/61.png)

:::caution
Loading a different eCard replaces your current workspace. If you have unsaved changes, a confirmation dialog asks whether you want to stay or discard your changes.
:::

![Confirmation dialog asking "You have unsaved changes" with Stay and Leave buttons](/img/card-designer/62.png)

---

## Unsaved changes protection

The Card Designer watches out for you in several ways:

- **Switching eCards** — A confirmation dialog appears if you try to load a different eCard while you have unsaved work
- **Signing out** — You'll be warned before logging out with unsaved changes
- **Closing the browser** — A native "Leave site?" prompt appears if you try to close the tab with unsaved work

![Navigation guard modal with amber warning, Stay button (purple), and Leave button (gray)](/img/card-designer/63.png)

:::tip
Save frequently using "Save to ELYSIUM" to keep your work safe. The Projects panel shows when each eCard was last saved.
:::
