# 导航

账户设置完成并打开应用后，你将看到三个主要界面，可通过从右向左滑动来切换。

## 界面 1：我的世界

在此界面，你可以查看已创建的世界，并开始创建新世界。界面底部有一个搜索图标，点击后可查看当前位置的地图。在地图上，你可以看到该区域内其他用户创建的所有场景。

## 界面 2：世界详情

在此界面，选择你的世界库中的某个世界后，你可以查看该世界的更多详细信息——包括元数据和场景列表。点击"编辑并查看场景"，将跳转至全局地图，地图上会显示一个代表你当前位置的标记点。向上滑动底部面板，然后左右滑动卡片即可浏览你的场景。如果你位于某个场景的地理围栏范围内，地图底部会出现一个绿色按钮，显示该场景的名称。点击该按钮即可进入 AR 模式。在此界面，你还可以创建新场景、编辑场景设置并发布场景。（注意：场景只有在所属世界已发布的情况下才能发布！否则，场景将保持草稿状态。）

## 界面 3：资产库

此界面是你的资产库，允许你从 iOS 设备中添加资产（对象、音频和图片）到库中，选择已上传的资产，或浏览 ELYSIUM 的免费资产合集。

## 你的资产库与添加新资产

成功设置账户后，首先需要将数字资产添加到你的库中。这些资产可以是 2D、3D、音频或视频格式。向右滑动到资产库页面，你将看到两个选项：从设备文件中浏览，或在线浏览合集。

如果选择从设备上传资产，你有两种方式：从文件或从照片库上传。选择所需资产后，按照提示为其命名、设置版权条款，以及可选的属性（如 URL 和元标签）。

如果选择在线浏览合集，请选择你想下载的合集来自文件或照片，最后同意版权条款即可。
