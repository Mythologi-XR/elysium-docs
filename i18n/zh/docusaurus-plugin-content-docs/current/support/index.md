# 联系我们

我们提供多种方式，让你从团队和不断壮大的创作者社区中获得支持与灵感。

## Discord

ELYSIUM 致力于建立一个创作者与玩家共同参与的强大网络，让社区知识能够在用户之间自由流通。我们建立社区的方式之一是运营 ELYSIUM Discord 服务器，成员可以在那里获取最新开发动态和常见问题解答，与其他用户分享创作，并参与线上活动和工作坊。

加入公开的 [ELYSIUM Discord 服务器](https://discord.gg/gYMKNYHJRJ)，与团队及其他 ELYSIUM 用户交流。
