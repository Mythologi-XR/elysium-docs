## Discord

ELYSIUM はクリエイターとプレイヤーの強力なネットワーク作りを目指しており、コミュニティの知識がユーザー間で自由に共有されるようにしています。このコミュニティを構築する方法の 1 つとして、Elysium Discord サーバーを運営しています。メンバーは最新の開発情報や FAQ にアクセスし、他のユーザーと作品を共有し、バーチャルイベントやワークショップに参加できます。

公開 [Elysium Discord サーバー](https://discord.gg/gYMKNYHJRJ) に参加して、チームや他の Elysium ユーザーとチャットしましょう。
