# 共有とプライバシー

Card Designer では、デザインを共有する 2 つの方法と、アクセス権を完全に管理する機能を提供しています。

## 2 つの共有方法

| | ローカルプレビュー | クラウド共有 |
|---|---|---|
| **方法** | エクスポートパネルの共有ボタン | 右サイドバーのプライバシーと共有パネル |
| **クロスデバイス対応** | なし（同じブラウザのみ） | あり（リンクを持つ誰でも） |
| **パスワード保護** | なし | オプション |
| **有効期限** | なし | 7 日、30 日、または 90 日（または無期限） |
| **引き受けとコラボレーション** | なし | オプション |

**ローカルプレビュー** は自分のデバイスでさっと確認するのに便利です。**クラウド共有** は他の人にリンクを送りたいときに使用してください。

---

## プライバシー設定

右サイドバーの **プライバシーと共有** パネル（シールドアイコン）では、eCard へのアクセス方法を制御できます。これらの設定を使用するには、クラウドに保存された eCard で[サインイン](./your-account)している必要があります。

![プライベート、共有を有効にする、コピーを許可する、引き受けを許可するの 4 つのプライバシートグルを表示する右サイドバーのプライバシーと共有パネル](/img/card-designer/64.png)

4 つのトグルでアクセスを制御します：

| トグル | デフォルト | 内容 |
|-------|---------|------|
| **プライベート** | オフ | eCard をパブリックな検索から非表示にします |
| **共有を有効にする** | オン | 共有リンクの作成を許可します |
| **コピーを許可する** | オフ | 受信者が eCard を複製して独自のコピーを編集できるようにします |
| **引き受けを許可する** | オフ | 受信者が eCard の完全な所有権を取得できるようにします |

変更は自動的に保存されます。トグルが即座に更新されます。

---

## クラウド共有リンクの作成

eCard を他の人と共有するには：

1. eCard が[クラウドに保存](./projects-and-cloud)されていることを確認してください
2. **プライバシーと共有** パネルを開いてください
3. **共有を有効にする** がオンになっていることを確認してください
4. 共有リンクの設定を選択してください：

### パスワード保護

| オプション | 説明 |
|----------|-----|
| **自動生成** | 安全な 9 文字のパスワードが自動的に作成されます |
| **カスタム** | 独自のパスワードを設定します（9 文字以上） |
| **なし** | パスワードなし — リンクを持つ誰でも閲覧できます |

### 有効期限

| オプション | 期限 |
|----------|-----|
| **7 日** | 作成から 1 週間後 |
| **30 日** | 作成から 1 か月後 |
| **90 日** | 作成から 3 か月後 |
| **無期限** | リンクが永続的に有効 |

5. **共有リンクを作成** をクリックしてください

![パスワードモードセレクター（自動生成、カスタム、なし）と有効期限ドロップダウン（7 日、30 日、90 日、無期限）を表示する共有リンク作成フォーム](/img/card-designer/65.png)

## 共有リンクの管理

アクティブな共有リンクは、プライバシーと共有パネルの展開可能なリストに表示されます。各リンクについて以下の操作が可能です：

- **URL をコピー** — コピーアイコンをクリックしてリンク全体をコピーします
- **有効期限を確認** — 残り時間のカウントダウンが表示されます（例：「15 日後に期限切れ」）
- **パスワードを確認** — リンクを作成またはパスワードをリセットした場合、共有できるようにパスワードが表示されます
- **パスワードをリセット** — 新しいパスワードを生成するか、カスタムパスワードを設定します（古いパスワードは無効になります）
- **取り消す** — リンクを永続的に無効化します

![コピー可能な URL、有効期限カウントダウン、パスワード表示、パスワードリセットボタン、取り消しボタンを含む展開された共有リンクリスト](/img/card-designer/66.png)

新しく作成されたリンクは自動的に展開され、URL とパスワードを簡単にコピーできます。

## 閲覧者に表示されるもの

誰かが共有リンクを開くと、デザインしたとおりにカードがレンダリングされたフルスクリーンの 3D ビューアーが表示されます：

- **3D カードプレビュー** — マテリアル、照明、エフェクトをフルで表示
- **情報バー** — カードのタイトル、カードタイプ、クリエイター名を表示
- **カメラプリセット** — 全 5 種類のプリセットが利用可能（ポートレート、ヒーロー、フロート、ドラマ、ショーケース）
- **カードの反転** — 閲覧者は表と裏を切り替えられます
- **オーディオ** — オーディオが添付されている場合、自動的に再生されます
- **リンクのコピー** — 閲覧者はツールバーから共有 URL をコピーできます

![カードのタイトル、タイプ、クリエイター名を表示する情報バーと浮かぶツールバーを含む 3D カードプレビューのクラウド共有プレイヤー](/img/card-designer/67.png)

閲覧者のデバイスが WebGL に対応していない場合、3D レンダリングの代わりに静的な画像プレビューが表示されます。

![3D レンダリングが利用できない場合のカードの静的画像プレビューを表示する WebGL フォールバック](/img/card-designer/68.png)

:::note
共有プレイヤーは読み取り専用です。閲覧者はデザインを変更できません。3D プレビューの閲覧、反転、操作のみが可能です。
:::

## パスワード保護された共有

パスワード保護付きリンクの場合、閲覧者はプレビューにアクセスする前にパスワード入力フォームが表示されます。

![パスワード入力フィールドと確認ボタンを含むパスワード確認フォーム](/img/card-designer/69.png)

**レート制限でカードを安全に保ちます：**
- 閲覧者は正しいパスワードを入力するまで **5 回** の試行ができます
- 5 回失敗すると、カウントダウンタイマーが表示された状態でアクセスが一時的にロックされます
- ロックアウト時には「クリエイターにお問い合わせください」というメッセージが表示されます

![失敗回数超過後のカウントダウンタイマーと「クリエイターへの連絡」メッセージを表示するレート制限ロックアウト画面](/img/card-designer/70.png)

## 引き受けとコラボレーション

プライバシー設定で **引き受けを許可する** または **コピーを許可する** を有効にした場合、閲覧者はただ見るだけでなく、より多くの操作が可能になります：

### 所有権の引き受け

**引き受けを許可する** がオンの場合、閲覧者には **引き受ける** ボタンが表示されます。引き受けると：

- eCard の完全な所有権が閲覧者に移転されます
- 共有リンクのパスワードが新しく生成されます（セキュリティのため）
- 元の所有者は編集アクセス権を失います

![所有権がユーザーに移転することを説明する黄色の警告スタイルの引き受け確認モーダル](/img/card-designer/71.png)

:::caution
引き受けは一方向の移転です。誰かが eCard を引き受けると、その人が新しい所有者となり、元の所有者は編集できなくなります。
:::

### 編集アクセス（コラボレーション）

**コピーを許可する** がオンの場合、閲覧者には編集アクセスをリクエストできる **閲覧** ボタンが表示されます。これにより：

- その人があなたの eCard の **コラボレーター** として追加されます
- **今すぐ編集**（Card Designer で開く）または **後で編集** を選択できます
- あなたは引き続き完全な制御権を持つ所有者のままです

![今すぐ編集と後で編集のオプションを表示する紫のスタイルの編集アクセス確認モーダル](/img/card-designer/72.png)

:::note
閲覧者が引き受けたり編集アクセスをリクエストしたりするには、[ELYSIUM アカウント](./your-account)が必要です。サインインしていない場合、サインイン画面が自動的に開きます。ログイン後に操作が自動的に再開されます。
:::

## ロール

共有システムでは 3 つのロールが認識されます：

| ロール | 対象者 | 表示内容 | 操作内容 |
|------|------|---------|---------|
| **オーナー** | あなた（クリエイター） | 「オーナー」バッジ | 完全な制御 — 編集、共有の管理、リンクの取り消し |
| **コラボレーター** | 編集アクセスを付与された人 | 「コラボレーター」バッジ | Card Designer で eCard を開いて編集 |
| **閲覧者** | それ以外のすべての人 | バッジなし | プレビューの閲覧、引き受けまたは編集アクセスのリクエスト（有効な場合） |

![情報バーのクリエイター名の横に表示されるオーナーバッジを表示する共有プレイヤー](/img/card-designer/73.png)

## ローカルプレビュー共有

エクスポートパネルとフルスクリーンツールバーの **共有** ボタンは、クイックローカルプレビューリンクを作成します。カードがブラウザのローカルストレージに保存され、即座にプレビューできるリンクが生成されます。

- エクスポートパネルまたはフルスクリーンツールバーの **共有** をクリックします
- コピーアイコンでリンクをコピーします（緑のチェックマークでコピー完了を確認）

:::caution
ローカルプレビューリンクは、作成した **同じブラウザとデバイス** でのみ機能します。ブラウザデータを消去すると永続的に削除されます。他の人と共有するには、プライバシーと共有パネルからクラウド共有をご利用ください。
:::
