# ELYSIUM Wallet

**The Ecosystem Identity Layer**

ELYSIUM Wallet (EIS-Wallet) is the unified identity and asset management layer for the ELYSIUM ecosystem. It provides a secure, portable identity that travels with users across all platform surfaces.
