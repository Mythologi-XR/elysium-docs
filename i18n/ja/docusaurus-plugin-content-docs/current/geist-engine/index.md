# GEIST Engine

**インタラクションエンジンとエクスペリエンスデザイン**

GEIST Engine は ELYSIUM の中核にあるインタラクションデザインシステムです。AR エクスペリエンス内の人、場所、デジタルオブジェクト、アクティビティの関係性を動かします。

## 近日公開予定

このセクションは現在開発中です。以下のドキュメントをお待ちください：

- 空間ロジックと近接イベント
- オブジェクトの動作とステートマシン
- 接続タイプ（人、場所、オブジェクト、アクティビティ）
- 魅力的なインタラクティブエクスペリエンスのデザイン
- クロスリアリティ接続（AR、IoT、ウェブ）
