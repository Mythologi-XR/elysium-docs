# 安装与账户设置

## 获取应用

ELYSIUM 目前仅支持 iOS 设备。你可以在 App Store 中搜索该应用，它归类于"图形与设计"类别。直接下载应用并在你的 iOS 设备（iPhone 或 iPad）上完成安装即可。

点击此处下载 iOS 版 ELYSIUM：

[![在 App Store 下载 ELYSIUM](/img/app-store-button.svg)](https://apps.apple.com/app/id1594239469)

## 创建账户

应用安装完成后，点击 ELYSIUM 图标打开应用。你将看到注册界面，可以在此创建账户。点击注册按钮并在下一个界面填写相关信息。你可以设置一个用户名，让其他用户知道你是谁以及你创建了哪些场景。注册时需要填写电子邮箱，并设置一个至少 8 位字符的密码。信息填写完成后，系统将发送一封确认邮件到你的邮箱。确认注册后，账户即设置完成，你可以开始探索 ELYSIUM 的世界。
