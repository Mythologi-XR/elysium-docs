# マテリアル、ライティング、エフェクト

## マテリアル仕上げを選ぶ

右パネルの **マテリアル** セクションで、カード表面の見た目を制御します。6 種類のプリセット仕上げから選択できます：

| 仕上げ | 最適な用途 |
|--------|----------|
| **フラット** | 無塗装のカード紙の質感 |
| **ホログラフィック** | レインボーホログラフィックトレーディングカード |
| **ゴールドフォイル** | プレミアムな金箔押しカード |
| **マット** | ソフトタッチのマット仕上げ |
| **グロッシー** | 高光沢の UV コーティング風 |
| **フロスト** | フロスト／ソフトグロスのハイブリッド |

![6 種類のマテリアルプリセットそれぞれで表示された同じカード——フラット、ホログラフィック、ゴールドフォイル、マット、グロッシー、フロスト](/img/card-designer/23.png)

完全に制御したい場合は **カスタム** を選択して、メタルネス、ラフネス、クリアコート、虹彩色を個別のスライダーで手動調整できます。

![メタルネス、ラフネス、クリアコート、虹彩色スライダーが表示されたカスタムマテリアルの選択](/img/card-designer/24.png)

### エミッシブグロー

**グロー** スライダーをオンにすると、カードが光を放射します——ネオンやバックライト効果に最適です。グローカラーを選んでエフェクトに色を付けることもできます。

![グローを有効にしたカードの 3D キャンバス上でのネオン/バックライト表示](/img/card-designer/25.png)

### エッジ仕上げ

カードのエッジ（表裏の間の薄い側面）には独自の仕上げがあります：**ソリッド**、**メタリック**、**ホログラフィック**。

![ソリッド、メタリック、ホログラフィックの仕上げを並べたカードエッジのクローズアップ](/img/card-designer/26.png)

## ライティングの設定

右パネルの **ライティング** セクションで 3 灯ライトのセットアップを制御します：

![キー/フィル/リムの強度スライダーとカラーピッカーが表示された右パネルのライティングセクション](/img/card-designer/27.png)

- **キーライト** — メインの光源。インタラクティブな方向球体をドラッグして向きを調整します。強度と色を変更できます。
- **フィルライト** — 強い影を和らげる柔らかい補助光
- **リムライト** — カードを背景から際立たせる後端の光

![ドラッグ可能なインジケータードットとクロスヘアが表示されたインタラクティブな方向球体](/img/card-designer/28.png)

強度スライダーと色を調整して、目指す雰囲気を作り出してください。

![デフォルトのライティングとドラマチックなカスタムライティングの同じカードの比較](/img/card-designer/29.png)

## ビジュアルエフェクト

右パネルで切り替え可能な 3 つのエフェクト：

- **ドロップシャドウ** — カードの下に柔らかな影を追加

  ![ドロップシャドウを有効にしたカード——下にソフトな影が表示された状態](/img/card-designer/30.png)

- **フロートアニメーション** — カードが穏やかに上下に浮かぶ

  ![フロートホバーサイクルの上端と下端でのカードの表示](/img/card-designer/31.png)

- **ライトパララックス** — カメラを動かすと微妙なハイライトのシフトが発生し、実際のホイルカードが光を捉える様子を模倣
