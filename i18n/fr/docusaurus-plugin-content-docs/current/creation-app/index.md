# Application de création ELYSIUM

**Création in situ et gestion de compte**

L'application de création ELYSIUM est l'outil mobile pour construire des expériences RA directement dans les espaces où elles seront vécues.

Cette section est en cours de développement. Revenez pour une documentation détaillée sur :

- Création de mondes et de scènes
- Placement d'objets et conception d'interactions
- Publication et partage depuis mobile
- Collaboration en temps réel
