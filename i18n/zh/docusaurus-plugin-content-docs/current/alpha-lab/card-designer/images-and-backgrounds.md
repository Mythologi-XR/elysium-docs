# 图片、背景与贴纸

## 向卡片添加图片

将图片直接拖放到左侧面板中标有标签的区域：

- **正面背景** — 卡片正面的完整背景图片
- **正面前景** — 叠加在背景之上的图片（标志、角色插画等）
- **背面图片** — 卡片背面的图片
- **品牌标志** — （仅限会员卡）你的品牌标志

![左侧面板显示标有标签的图片拖放区域，带有占位符状态](/img/card-designer/13.png)

将图片拖到拖放区域时，注意**紫色高亮**效果——这表示该区域已准备好接受你的文件。

![将图片拖到拖放区域时的紫色高亮效果](/img/card-designer/14.png)

上传前景图片后，使用**偏移**和**缩放**滑块来精确调整其位置和大小。

![上传前景图片后显示前景偏移 X/Y 和缩放滑块](/img/card-designer/15.png)

支持格式：JPEG、PNG 和 WebP。

## 选择背景

左侧面板中的背景选择器提供多种选项：

![背景选择器区域，显示类型选择器——纯色、渐变、HDRI、图片、透明](/img/card-designer/16.png)

- **纯色** — 带有不透明度控制的单一颜色
- **渐变** — 带有角度和不透明度的双色渐变
- **HDRI 预设** — 从工作室、日落、仓库或森林环境中选择，以获得逼真的反射效果

  ![四个 HDRI 预设缩略图——工作室、日落、仓库、森林](/img/card-designer/17.png)

- **HDRI 自定义** — 上传你自己的 `.hdr` 或 `.exr` 环境文件
- **图片** — 使用任意图片作为平面背景
- **透明** — 无背景（适合导出干净的抠图）

使用 HDRI 背景时，**旋转**滑块可让你旋转环境，将光源和反射精确定位到你想要的位置。

![同一张卡片在两种不同 HDRI 旋转角度下显示反射变化](/img/card-designer/18.png)

## 添加渐变

两种卡片类型都支持在正面和背面添加渐变叠加层。在左侧面板的内容编辑器区域：

![渐变区域，显示起始/终止颜色选择器、不透明度、角度和链接/取消链接切换按钮](/img/card-designer/19.png)

- 为渐变选择**起始**和**终止**颜色
- 调整**不透明度**和**角度**
- 默认情况下，正面和背面的渐变处于**链接**状态——对一面的更改会镜像到另一面。点击链条图标可取消链接，进行独立控制。
- 使用**翻转**图标水平镜像背面渐变

![链接渐变（正背面相同）与取消链接（正背面不同）对比](/img/card-designer/20.png)

## 添加贴纸

想在卡片上叠加额外图片？**贴纸**区域允许每张卡片添加最多 8 个自定义图片贴纸。

![贴纸区域，添加了多个贴纸，显示标题、面朝切换、缩放、偏移和拖动手柄](/img/card-designer/21.png)

对于每个贴纸，你可以控制：
- 它显示在哪一**面**（正面或背面）
- 其**缩放比例**（20% 至 200%）
- 使用偏移滑块调整其**位置**
- 使用拖动手柄对列表重新排序，控制其**层级顺序**

贴纸位于所有其他内容之上、材质效果之下，因此看起来像是自然"印刷"在卡片上。

![3D 画布显示卡片正面渲染的自定义贴纸](/img/card-designer/22.png)
