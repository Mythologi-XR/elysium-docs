# ELYSIUM 简介

**以临场感为驱动力。赋予设计以意义。共同游玩。**

ELYSIUM 是在现实世界中创造沉浸式数字体验的全新范式。

它是一套创作工具包与云平台，让任何人都能以实时方式在增强现实中构建属于自己的世界，并与他人分享。

通过我们独特的无代码交互工具体系，所有用户都能成为真正意义上的创作者，在真实的空间与地点，通过互动式旅程，构建_他们自己_的故事世界。在建立连接之外，我们的交互设计引擎——GEIST Engine——让你在人、地点、数字内容与活动之间创建_关系_。

创作时，你所见即你的观众所见。构建过程中，你的作品可随时发布并分享。

这是面向空间互联网的"所见即所得"（WYSIWYG）编辑器。
