# REALITY BRIDGE

**Infrastructure matérielle**

REALITY BRIDGE est le système de modules matériels qui connecte le monde physique à la plateforme ELYSIUM. Ces appareils compatibles IoT servent d'ancres, de déclencheurs et de capteurs environnementaux pour les expériences RA.
