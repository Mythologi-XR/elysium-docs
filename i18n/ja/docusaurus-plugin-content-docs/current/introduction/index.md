# ELYSIUM へようこそ

**存在感が力。意味をデザインする。ともに遊ぶ。**

ELYSIUM は、現実世界に没入型デジタル体験を生み出すための新しいパラダイムです。

クリエイションツールキットとクラウドプラットフォームを兼ね備えており、誰でもリアルタイムで拡張現実の世界を構築し、他者と共有することができます。

独自のノーコードインタラクティブツールを通じて、すべてのユーザーが自らのクリエイターになれます。現実の空間や場所を舞台にしたインタラクティブな旅を通じて、_自分自身_ のストーリーを語る世界を構築できます。さらに、私たちのインタラクションデザインエンジン「GEIST Engine」は、人・場所・デジタル・活動の _関係性_ を創り出します。

制作中は、視聴者が見るのとまったく同じ画面を確認できます。そして作り上げた体験は、即座に公開して共有できます。

空間ウェブのための WYSIWYG（見たまま編集）エディター。
