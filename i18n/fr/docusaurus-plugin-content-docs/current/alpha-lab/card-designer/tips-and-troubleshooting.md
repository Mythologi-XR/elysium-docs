# Conseils et dépannage

## Raccourcis clavier

| Touche | Ce qu'elle fait |
|--------|----------------|
| **1 - 5** | Changer de préréglage de caméra |
| **F** | Retourner la carte |
| **Échap** | Quitter l'aperçu plein écran / ouvrir la boîte de dialogue de sortie |
| **Double-clic** | Retourner la carte (alternative) |

:::tip
Les raccourcis clavier pour les préréglages de caméra sont mis en pause lorsque vous tapez dans un champ de texte, afin de ne pas changer accidentellement de vue pendant la saisie de contenu.
:::

## Dépannage

### Mes images ne s'affichent pas
- Assurez-vous d'utiliser des fichiers JPEG, PNG ou WebP
- Déposez l'image directement sur la zone étiquetée — attendez la surbrillance violette avant de relâcher
- Si l'image est très grande, essayez de la redimensionner avant de la téléverser

### Mon export est flou
- Augmentez le multiplicateur de résolution à 3x ou 4x
- Utilisez des images sources à plus haute résolution
- Pour les impressions les plus nettes, utilisez le mode **Faces de carte** plutôt que le mode **Scène 3D**

### L'aperçu 3D ne se charge pas
- Assurez-vous que l'accélération matérielle est activée dans les paramètres de votre navigateur
- Essayez un autre navigateur (Chrome ou Edge fonctionnent le mieux)
- Mettez à jour vos pilotes graphiques

### Mon lien de partage ne fonctionne pas pour quelqu'un d'autre

**Pour les liens d'aperçu local** (créés avec le bouton Partager dans le panneau Export) : ceux-ci ne fonctionnent que sur le même navigateur et appareil où ils ont été créés. Utilisez plutôt le [partage cloud](./sharing-and-privacy) pour partager des liens qui fonctionnent pour n'importe qui.

**Pour les liens de partage cloud**, vérifiez ces causes courantes :
- **"Expiré"** — Le lien a dépassé sa date d'expiration. Demandez au créateur d'en générer un nouveau
- **"Révoqué"** — Le créateur a définitivement désactivé ce lien
- **"Partage désactivé"** — Le créateur a désactivé le partage pour cette eCarte
- **"Trop de tentatives"** — Vous avez dépassé 5 tentatives de mot de passe. Attendez que le compte à rebours se réinitialise

### Je ne peux pas me connecter ou créer un compte
- Vérifiez votre e-mail et votre mot de passe — les mots de passe sont sensibles à la casse
- Si votre session a expiré, reconnectez-vous simplement (les sessions se renouvellent automatiquement mais peuvent expirer après une longue inactivité)
- Lors de la création d'un compte, assurez-vous que votre mot de passe comporte au moins 8 caractères et que votre e-mail n'est pas déjà utilisé

### Ma sauvegarde cloud ne fonctionne pas
- **Le téléversement semble bloqué** — Vérifiez votre connexion internet et réessayez
- **"La description est requise"** — Le champ de description est obligatoire — remplissez-le avant d'enregistrer
- **L'eCarte n'apparaît pas dans le panneau Projets** — Essayez de la rechercher, ou assurez-vous d'être connecté au bon compte

### L'export GLB affiche des erreurs

![Panneau d'export montrant un état d'erreur rouge avec une liste d'erreurs développée après un échec de validation GLB](/img/card-designer/45.png)

- Essayez d'activer **Fusionner les textures** dans les paramètres d'export

![Panneau d'export montrant un avertissement ambré "Valide avec avertissements" après l'export GLB](/img/card-designer/46.png)

- Si les erreurs persistent, rechargez la page, re-téléversez vos images et réessayez
