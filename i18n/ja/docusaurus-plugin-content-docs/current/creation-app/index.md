# ELYSIUM Creation App

**現場での制作とアカウント管理**

ELYSIUM Creation App は、AR エクスペリエンスが体験される空間で直接 AR コンテンツを構築するためのモバイルファーストのツールです。

## 近日公開予定

このセクションは現在開発中です。以下のドキュメントをお待ちください：

- 現場での AR 制作ワークフロー
- アカウントのセットアップと管理
- ワールドとシーンの作成
- アセットの配置とインタラクションデザイン
- モバイルからの公開と共有
- リアルタイムコラボレーション
