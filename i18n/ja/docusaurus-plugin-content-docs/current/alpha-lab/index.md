# ALPHA Lab

**実験的な機能とクリエイティブツール**

ALPHA Lab は、ELYSIUM プラットフォームのために開発されている最先端のツールと実験的な機能の場所です。これらのツールは現在も活発に開発が進められており、改善に伴って進化していきます。

## 利用可能なツール

- **[Card Designer](/alpha-lab/card-designer)** — 3D カードデザインスタジオで魅力的な AR カードやロイヤルティカードをデザイン
