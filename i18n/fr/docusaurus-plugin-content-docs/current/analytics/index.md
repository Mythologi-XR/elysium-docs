# ELYSIUM Analytics

**Suite d'analyse comportementale en temps réel**

ELYSIUM Analytics est une suite d'analyse dédiée, conçue pour capturer, traiter et mettre en valeur les données comportementales issues des expériences RA — structurées pour l'analyse humaine et la consommation par l'IA.

## Bientôt disponible

Cette section est en cours de développement. Revenez pour une documentation détaillée sur :

- Taxonomie des événements et modèle de données
- Pipelines de données en temps réel
- Analyses spatiales et cartes thermiques
- Indicateurs de comportement utilisateur et d'engagement
- Formats d'export de données prêts pour l'IA
- Outils de tableau de bord et de reporting
- Intégration avec les événements du moteur GEIST
