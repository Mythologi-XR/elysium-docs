# 什么是 Card Designer？

直接在浏览器中设计精美的 AR 和会员卡——无需任何设计软件。Card Designer 提供实时 3D 预览，在你构建卡片的同时即可看到最终效果，所见即所得。

![Card Designer 工作区，3D 画布上可见已完成的卡片设计](/img/card-designer/1.png)

## 可以制作什么？

两种卡片类型：

- **AR 卡片** — 竖向收藏卡（类似集换式卡牌），扫描后可触发增强现实体验。非常适合互动内容、数字收藏品或品牌 AR 推广活动。

  ![已完成的 AR 卡片——竖向方向](/img/card-designer/2.png)

- **会员卡** — 横向会员积分卡，配有二维码、条形码和自定义货币面额。适用于世界忠诚度计划和品牌体验。

  ![已完成的会员卡——横向方向，带有二维码和品牌标识](/img/card-designer/3.png)

## 云端功能

创建免费的 [ELYSIUM 账户](./your-account)以解锁：

- **云端保存** — 保存设计作品，并从任意设备访问
- **跨设备分享** — 与任何人分享受密码保护的链接
- **协作** — 允许他人查看、复制或认领你的卡片

即使没有账户，所有功能也可在本地使用——随时可以创建账户。

## 开始前的准备

你只需要一款现代浏览器（Chrome、Firefox、Edge 或 Safari），其他一切都在浏览器中运行。为获得最佳体验，建议使用屏幕宽度至少为 1024px 的桌面或笔记本电脑。
