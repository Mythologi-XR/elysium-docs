# モード

## 遊ぶ＋体験する（デモモード）

アプリをインストールしているすべてのユーザーがデモモードを利用できます。このモードでは ELYSIUM の基本的な機能を体験できます。無料アセットの限定ライブラリから選んで、アセットを物理的な場所に配置する方法を探索できます。このモードでは、シーンの作成・保存・共有はできません。

## 探索＋発見（プレイヤーモード）

このモードでは、ジオロケーションシーンを活用して作られた魅力的な AR ワールドを閲覧・体験できます。プレイヤーとして、ユーザーが作成した AR シーンを閲覧し、その世界に没入できます。さまざまな場所でリアルタイムに展開するライブストーリーやインタラクティブなアドベンチャーを体験しましょう。思い出の体験を写真や動画として記録し、他の人と共有できます。また、登録することでプレイの進行状況とインタラクションを保存でき、いつでも続きから始められます。探索・体験・存在感の力に満ちた忘れられない AR アドベンチャーへ出発しましょう。

## 制作＋共有（クリエイターモード）

このモードでは、AR 体験とデジタルアセットのプライベートライブラリを所有・管理できます。無限の可能性の中で没入型のワールドを構築し、ユニークで魅力的な環境を創り出しましょう。ストーリーをデザインし、魅力的な体験を作ることで、想像力を現実に変え、視聴者を魅了できます。インタラクティビティやゲーミフィケーション要素を加えて、AR ワールドをさらにスリリングで魅力的なものにしましょう。傑作が完成したら、簡単に視聴者と共有して、あなたの作品を通じた素晴らしいジャーニーへ旅立たせましょう。アーティスト、ストーリーテラー、ゲームデザイナーなど、クリエイターモードはあなたの才能を発揮し、想像力豊かな作品を世界と共有するためのツールとプラットフォームを提供します。
