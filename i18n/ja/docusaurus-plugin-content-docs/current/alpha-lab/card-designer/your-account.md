# ELYSIUM アカウント

Card Designer はアカウントなしでもご利用いただけます。すべての機能がブラウザ上でローカルで動作します。ただし、無料の ELYSIUM アカウントを作成すると、強力なクラウド機能が使えるようになります。

- **クラウドへの保存** — どのデバイスからでもデザインにアクセスできます
- **クロスデバイス共有** — どこからでも誰でもアクセスできる共有リンクを作成できます
- **コラボレーション** — カードの閲覧、コピー、または引き受けを他のユーザーに許可できます
- **プロジェクト管理** — すべての eCard を一か所で閲覧・検索・整理できます

## アカウントの作成

1. サイドバーの **「ELYSIUM に保存」** をクリックするか、プロジェクトパネルの **「アカウントを作成」** をクリックしてください

![「サインイン」と「アカウントを作成」のオプションを表示した認証モーダル画面](/img/card-designer/47.png)

2. 登録フローは 3 つのステップで完了します：
   - **ユーザー情報** — 名前、メールアドレス、パスワード（8 文字以上）を入力してください
   - **ユーザー名の選択** — メールアドレスから自動生成されますが、お好みのものに変更することもできます

![名前、メールアドレス、パスワードフィールドを含む登録フォーム](/img/card-designer/48.png)

![自動生成されたユーザー名と編集オプションを表示するユーザー名選択ステップ](/img/card-designer/49.png)

3. 登録後、自動的にログインされ、すぐに利用を開始できます

:::tip
先にデザインを始めて、後でアカウントを作成することもできます。クラウドに保存する準備ができたら「ELYSIUM に保存」をクリックしてください。
:::

## サインイン

1. プロジェクトパネルの **「サインイン」** をクリックするか、保存などのクラウド操作を実行してください
2. メールアドレスとパスワードを入力してください
3. **サインイン** をクリックしてください

![メールアドレスとパスワードフィールドを含むサインインフォーム](/img/card-designer/50.png)

セッションは 60 分間有効で、バックグラウンドで自動的に更新されます。ブラウザを閉じて後で戻ってきた場合も、7 日以上離れていない限りサインイン状態が維持されます。

## サインアウト

サイドバー下部の **アカウント名** をクリックしてサインアウトします。

![認証済みユーザーの名前とサインアウトオプションを表示するサイドバーアクションエリア](/img/card-designer/51.png)

- **未保存の変更** がある場合、先に保存するかどうかを確認するダイアログが表示されます

![未保存の変更がある状態でサインアウトしようとした際に表示される警告ダイアログ](/img/card-designer/52.png)

- サインアウトしても、ローカルの自動保存データは **削除されません** — 作業中のデザインはブラウザに残ります

:::note
ローカルの自動保存データやダウンロードした `.ecard` ファイルは、クラウドの保存データとは完全に独立しています。サインアウトしてもそれらには影響しません。
:::
