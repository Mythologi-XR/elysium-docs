# 材质、光照与特效

## 选择材质效果

右侧面板的**材质**区域控制卡片表面的外观。从六种现成效果中选择：

| 效果 | 适用场景 |
|------|----------|
| **哑光** | 无涂层卡纸外观 |
| **全息** | 彩虹全息集换式卡牌 |
| **金箔** | 高端金色印压卡片 |
| **磨砂** | 柔触磨砂效果 |
| **光泽** | 高光 UV 涂层外观 |
| **磨砂光** | 磨砂/柔光混合效果 |

![同一张卡片展示 6 种材质预设——哑光、全息、金箔、磨砂、光泽、磨砂光](/img/card-designer/23.png)

想要完全控制？选择**自定义**，使用独立滑块手动调整金属感、粗糙度、清漆和彩虹光效。

![选择自定义材质后显示金属感、粗糙度、清漆和彩虹光效滑块](/img/card-designer/24.png)

### 发光效果

开启**发光**滑块，让卡片发出光芒——非常适合霓虹灯或背光效果。还可以选择发光颜色为效果着色。

![启用发光后的卡片，在 3D 画布上呈现霓虹/背光外观](/img/card-designer/25.png)

### 边缘效果

卡片的边缘（正背面之间的薄侧面）拥有独立效果：**实心**、**金属**或**全息**。

![卡片边缘近景，并排显示实心、金属和全息效果](/img/card-designer/26.png)

## 设置光照

右侧面板的**光照**区域控制三光源设置：

![右侧面板的光照区域，显示主光/补光/轮廓光强度滑块和颜色选择器](/img/card-designer/27.png)

- **主光** — 你的主要光源。拖动交互式方向球来瞄准它，可调整强度和颜色。
- **补光** — 一个较柔和的辅助光源，用于减少强烈阴影
- **轮廓光** — 背部边缘光，使卡片从背景中凸显出来

![交互式方向球，显示可拖动的指示点和准星](/img/card-designer/28.png)

调整强度滑块和颜色，打造你想要的氛围。

![同一张卡片在默认光照与自定义戏剧性光照设置下的对比](/img/card-designer/29.png)

## 视觉特效

右侧面板中的三种可切换效果：

- **投影** — 在卡片下方添加柔和阴影

  ![启用投影后的卡片，显示下方柔和阴影](/img/card-designer/30.png)

- **悬浮动画** — 使卡片轻柔地上下漂浮

  ![卡片在悬浮循环顶部和底部的状态](/img/card-designer/31.png)

- **光线视差** — 随着摄像机移动，高光会微妙地偏移，模拟真实箔卡捕捉光线的方式
