# Card Content

The left panel's content editor changes depending on which card type you chose during setup.

## AR Card content

The AR Content Editor lets you configure what happens when your card is scanned in AR:

![AR Content Editor with fields filled in — HTML overlay, audio, logo, buttons, and metadata](/img/card-designer/32.png)

- **Front/Back HTML overlays** — Rich text or HTML content displayed over each face
- **Audio** — Upload an MP3 that plays during the AR experience
- **Brand logo** — Your logo with adjustable scale
- **AR Buttons** — Up to 4 interactive buttons (top, right, bottom, left) that can open links, trigger AR traces, dismiss overlays, or flip the card

  ![AR Buttons section showing button labels, action type dropdown, and action value fields](/img/card-designer/33.png)

- **Metadata** — Card title, AR ID, series name, edition size, and collection order

Each element has a visibility toggle so you can show or hide it without deleting the content.

![Eye-icon visibility toggles next to content elements](/img/card-designer/34.png)

## Loyalty Card content

The Loyalty Content Editor has fields specific to loyalty programs:

![Loyalty Content Editor with world name, description, QR code, barcode, and currencies filled in](/img/card-designer/35.png)

- **World name, description, and tagline**
- **Brand logo** with scale control
- **QR code and barcode** — Drag and drop your code images
- **Currency denominations** — Add up to 10 custom currencies (like "100 LUMEN" or "5 ORBS")

  ![Currency section showing denominations with name and symbol fields](/img/card-designer/36.png)
