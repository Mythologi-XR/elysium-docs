# ヒントとトラブルシューティング

## キーボードショートカット

| キー | 操作 |
|-----|------|
| **1 〜 5** | カメラプリセットの切り替え |
| **F** | カードの反転 |
| **Esc** | フルスクリーンプレビューを終了 / 終了ダイアログを開く |
| **ダブルクリック** | カードの反転（別の方法） |

:::tip
テキストフィールドに入力中はカメラプリセットのキーボードショートカットが一時停止されるため、コンテンツ入力中に誤ってビューが切り替わることはありません。
:::

## トラブルシューティング

### 画像が表示されない

- JPEG、PNG、または WebP ファイルを使用していることを確認してください
- 放す前に紫のハイライトが表示されるのを確認しながら、ラベル付きのゾーンに画像を直接ドロップしてください
- 画像のサイズが非常に大きい場合は、アップロード前にリサイズを試してみてください

### エクスポートがぼやけて見える

- 解像度の倍率を 3x または 4x に上げてください
- より高解像度のソース画像を使用してください
- 最もシャープな印刷結果を得るには、**3D シーン** モードではなく **カード面** モードを使用してください

### 3D プレビューが読み込まれない

- ブラウザの設定でハードウェアアクセラレーションが有効になっていることを確認してください
- 別のブラウザを試してください（Chrome または Edge が最適です）
- グラフィックスドライバーを更新してください

### 共有リンクが他のユーザーで機能しない

**ローカルプレビューリンク**（エクスポートパネルの共有ボタンで作成したもの）：これらは作成した同じブラウザとデバイスでのみ機能します。誰でも使えるリンクを共有するには、代わりに[クラウド共有](./sharing-and-privacy)を使用してください。

**クラウド共有リンク**の場合、以下の一般的な原因を確認してください：
- **「期限切れ」** — リンクが有効期限を過ぎています。クリエイターに新しいリンクを発行してもらってください
- **「取り消し済み」** — クリエイターがこのリンクを永続的に無効化しています
- **「共有が無効」** — クリエイターがこの eCard の共有をオフにしています
- **「試行回数超過」** — パスワードを 5 回超えて試行しています。カウントダウンタイマーがリセットされるまでお待ちください

### サインインまたはアカウントの作成ができない

- メールアドレスとパスワードを確認してください（パスワードは大文字と小文字を区別します）
- セッションが期限切れの場合は、再度サインインしてください（セッションは自動的に更新されますが、長時間非アクティブな状態が続くとタイムアウトする場合があります）
- アカウントを作成する際は、パスワードが 8 文字以上であること、そのメールアドレスがまだ使用されていないことを確認してください

### クラウド保存が機能しない

- **アップロードが止まっているように見える** — インターネット接続を確認して、再試行してください
- **「説明が必要です」** — 説明フィールドは必須です。保存前に入力してください
- **プロジェクトパネルに eCard が表示されない** — 検索してみるか、正しいアカウントでサインインしていることを確認してください

### GLB エクスポートでエラーが表示される

![GLB 検証失敗後のエラーリストが展開された赤いエラー状態のエクスポートパネル](/img/card-designer/45.png)

- エクスポート設定で **テクスチャをベイク** をオンにしてみてください

![GLB エクスポート後の「警告はあるが有効」を示すアンバー色の警告のエクスポートパネル](/img/card-designer/46.png)

- エラーが続く場合は、ページを再読み込みし、画像を再アップロードして、再試行してください
