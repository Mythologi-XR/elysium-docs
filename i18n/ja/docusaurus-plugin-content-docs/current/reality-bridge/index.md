# REALITY BRIDGE

**ハードウェアインフラ**

REALITY BRIDGE は、物理世界を ELYSIUM プラットフォームに接続するハードウェアモジュールシステムです。これらの IoT 対応デバイスは、AR エクスペリエンスのアンカー、トリガー、環境センサーとして機能します。
