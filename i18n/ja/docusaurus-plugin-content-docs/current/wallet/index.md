# ELYSIUM Wallet

**エコシステムアイデンティティレイヤー**

ELYSIUM Wallet（EIS-Wallet）は、ELYSIUM エコシステムの統合アイデンティティおよびアセット管理レイヤーです。すべてのプラットフォームサーフェス全体でユーザーと一緒に移動する、安全でポータブルなアイデンティティを提供します。
