# 互动功能

ELYSIUM 提供丰富的互动功能，并持续开发让体验更具共融性和沉浸感的新方式。目前 ELYSIUM 主要提供以下互动功能：

- **拖放式布置：** 为使构建世界尽可能直观，ELYSIUM 允许创作者通过从资产库中拖拽素材，轻松将其放置在设备摄像头所呈现的精确位置，从而搭建场景。
- **沉浸式 3D 资产：** 一旦将 3D 资产放置在某个位置，玩家和创作者就可以对物体进行缩放、移位，并像与实体物品互动一样与之互动。
- **方向性空间音频：** 音频资产同样可以放置在精确位置，类似于 3D 资产。玩家可以根据自身与资产的距离，调整声音的音量和方向。
- **互操作性：** ELYSIUM 支持将数字资产与外部硬件集成，使按下虚拟按钮能够产生物理反应。例如，与灯泡相连的虚拟资产可以通过按压来控制灯的开关。

## 互操作性

互操作性正在作为 ELYSIUM 的一项主要互动功能持续整合开发。为拓展 AR 在现实世界中的可能性，我们的团队一直致力于构建资产与硬件之间的集成系统。通过这一功能，在 ELYSIUM 中放置的数字资产可以与实体电子设备相关联，仅凭数字触发器即可驱动硬件产生响应。

## 平台支持

目前，ELYSIUM 仅支持 iOS 设备，但我们的团队正在努力扩大软件的覆盖范围，以便 ELYSIUM 能够在更多操作系统上运行。
