# Tips and Troubleshooting

## Keyboard shortcuts

| Key | What it does |
|-----|-------------|
| **1 - 5** | Switch camera preset |
| **F** | Flip card |
| **Esc** | Exit fullscreen preview / open exit dialog |
| **Double-click** | Flip card (alternative) |

:::tip
Keyboard shortcuts for camera presets are paused when you're typing in a text field, so you won't accidentally switch views while entering content.
:::

## Troubleshooting

### My images aren't showing up
- Make sure you're using JPEG, PNG, or WebP files
- Drop the image directly onto the labeled zone — watch for the purple highlight before releasing
- If the image is very large, try resizing it before uploading

### My export looks blurry
- Increase the resolution multiplier to 3x or 4x
- Use higher-resolution source images
- For the sharpest prints, use **Card Faces** mode instead of **3D Scene** mode

### The 3D preview isn't loading
- Make sure hardware acceleration is enabled in your browser settings
- Try a different browser (Chrome or Edge work best)
- Update your graphics drivers

### My share link doesn't work for someone else
This is expected in the current version — share links are stored locally in your browser and only work on the same device. Cross-device sharing is planned for a future update.

### GLB export shows errors

![Export panel showing a red error state with expanded error list after a failed GLB validation](/img/card-designer/45.png)

- Try turning on **Bake Textures** in the export settings

![Export panel showing an amber warning with "Valid with warnings" after GLB export](/img/card-designer/46.png)

- If errors persist, reload the page, re-upload your images, and try again
