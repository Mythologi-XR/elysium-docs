# Card Designer とは？

ブラウザ上で魅力的な AR カードやロイヤルティカードをデザインできます——デザインソフトは不要です。Card Designer では、カードを作成しながらリアルタイムで 3D プレビューが表示されるため、見た通りの仕上がりになります。

![Card Designer のワークスペースと完成したカードデザインが 3D キャンバスに表示されている画面](/img/card-designer/1.png)

## 何が作れる？

2 種類のカードを作成できます：

- **AR カード** — AR 体験を起動するスキャン可能な縦型コレクタブルカード（トレーディングカード風）。インタラクティブコンテンツ、デジタルコレクタブル、ブランド AR キャンペーンに最適です。

  ![完成した AR カード——縦向きの向き](/img/card-designer/2.png)

- **ロイヤルティカード** — QR コード、バーコード、カスタム通貨単位を備えた横型のロイヤルティプログラムカード。ワールドのロイヤルティプログラムやブランド体験に最適です。

  ![完成したロイヤルティカード——QR コードとブランドロゴ付きの横向きの向き](/img/card-designer/3.png)

## クラウド機能

無料の [ELYSIUM アカウント](./your-account) を作成してこれらの機能を解放：

- **クラウド保存** — デザインを保存して、どのデバイスからでもアクセス
- **クロスデバイス共有** — パスワード保護付きリンクを誰とでも共有
- **コラボレーション** — 他のユーザーがカードを閲覧・コピー・クレームできるように設定

アカウントなしでもローカルですべての機能が使えます——後からアカウントを作成することもできます。

## 始める前に

モダンなウェブブラウザ（Chrome、Firefox、Edge、または Safari）があれば大丈夫です。すべてブラウザ上で動作します。最良の体験のために、画面幅が 1024px 以上のデスクトップまたはノートパソコンを使用してください。
