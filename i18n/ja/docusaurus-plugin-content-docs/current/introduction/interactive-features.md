# インタラクティブ機能

ELYSIUM は豊富なインタラクティブ機能を提供しており、体験をよりコミュニティ的・没入的にする新しい方法を継続的に開発しています。ELYSIUM が現在提供している主なインタラクティブ機能は以下のとおりです：

- **ドラッグ＆ドロップ配置：** ワールドの構築をできる限り直感的にするため、ELYSIUM ではクリエイターがライブラリからアセットをドラッグし、デバイスのカメラで確認しながら正確な場所に配置できます。
- **没入型 3D アセット：** 3D アセットをある場所に配置すると、プレイヤーとクリエイターはそのオブジェクトをまるでそこに実在しているかのようにサイズ変更・再配置・インタラクションできます。
- **方向性空間オーディオ：** オーディオアセットも 3D アセットと同様に正確な場所に配置できます。このアセットにより、プレイヤーはアセットへの近接度に応じて音の音量や方向を変えることができます。
- **相互運用性：** ELYSIUM ではデジタルアセットを外部ハードウェアと連携させることができ、バーチャルボタンを押すと現実世界で物理的な反応を引き起こします。たとえば、電球と接続されたバーチャルアセットを押すことで、照明をオン・オフできます。

## 相互運用性

相互運用性は現在、ELYSIUM の主要なインタラクティブ機能として統合が進められています。AR の現実世界での可能性を発展させる取り組みの一環として、チームはアセットとハードウェアを連携させた統合システムを開発しています。この機能により、ELYSIUM 内に配置されたデジタルアセットを物理的な電子デバイスと連携させ、デジタルアクチュエーターのみを使ってハードウェアからの応答を引き出すことができます。

## プラットフォーム対応状況

現在、ELYSIUM は iOS デバイスのみで利用可能ですが、チームはソフトウェアの対応範囲を拡大すべく鋭意開発中です。さまざまなオペレーティングシステムで ELYSIUM が動作できるよう取り組んでいます。
