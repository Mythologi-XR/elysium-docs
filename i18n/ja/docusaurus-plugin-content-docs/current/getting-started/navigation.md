# ナビゲーション

アカウントを設定してアプリを開くと、左右にスワイプしてアクセスできる 3 つのメイン画面が表示されます。

## 画面 1：マイワールド

この画面では、すでに作成したワールドを確認したり、新しいワールドを作成したりできます。画面下部の検索アイコンをタップすると、現在地のマップが表示されます。このマップでは、あなたのエリアで作成された他のシーンも確認できます。

## 画面 2：ワールドの詳細

この画面では、ライブラリから選択したワールドの詳細情報（メタデータやシーン一覧など）を確認できます。「シーンを編集・表示」をタップすると、現在地を示すピンが表示されたグローバルマップに移動します。ワールド内のすべてのシーンを表示するには、下部パネルを上にスワイプしてから、カードをスワイプしてシーンを切り替えてください。シーンのジオフェンス内にいる場合、マップ下部のボタンにシーン名が緑色で表示されます。そのボタンをタップすると AR に入ることができます。この画面では、新しいシーンの作成、シーン設定の編集、シーンの公開も行えます。（注意：シーンを公開するには、親ワールドが公開されている必要があります。そうでない場合、シーンは下書きのままになります。）

## 画面 3：アセットライブラリ

この画面はアセットライブラリです。iOS デバイスからアセット（オブジェクト、オーディオ、画像）をライブラリに追加したり、すでにアップロードしたアセットを選択したり、ELYSIUM の無料アセットコレクションを閲覧したりできます。

## アセットライブラリと新しいアセットの追加

アカウントの設定が完了したら、まずデジタルアセットをライブラリに用意しましょう。2D、3D、オーディオ、動画など様々な形式に対応しています。ライブラリページに右スワイプすると、2 つの選択肢が表示されます：デバイスのファイルを閲覧するか、オンラインのコレクションを閲覧するかです。

デバイスからアセットをアップロードする場合、ファイルから、または写真ライブラリからの 2 種類の方法があります。アセットを選択し、名前、著作権条件、URL やメタタグなどの任意属性を設定する画面を進んでください。

オンラインコレクションを閲覧する場合は、ダウンロードしたいコレクションを選択し、著作権条件に同意してください。
