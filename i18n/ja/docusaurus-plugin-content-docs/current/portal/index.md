# ELYSIUM Portal

**クリエイティブツール、ダッシュボード、プレイヤーハブ**

ELYSIUM Portal は、クリエイターとプレイヤーのためのウェブベースのコントロールセンターです。ワールドの管理、アセットのインポート、アナリティクスの追跡、オーディエンスとのつながりが可能です。
