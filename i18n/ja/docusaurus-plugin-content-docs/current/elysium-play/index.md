# ELYSIUM PLAY

**クロスプラットフォームプレイの概要とデザイン**

ELYSIUM PLAY は、モバイル、ウェブ、ウェアラブルインターフェース全体で AR ワールドをディスカバー、参加、インタラクションできるクロスプラットフォームエクスペリエンスレイヤーです。
