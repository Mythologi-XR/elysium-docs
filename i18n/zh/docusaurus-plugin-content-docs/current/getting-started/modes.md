# 模式

## 游玩 + 探索（演示模式）

任何在设备上安装了应用的用户都可以使用演示模式。该模式允许用户体验 ELYSIUM 的基本功能。用户可以从有限的免费资产库中选择素材，探索如何将资产放置到真实位置。在此模式下，用户无法创建、保存或分享场景。

## 探索 + 发现（玩家模式）

通过此模式，你可以浏览和探索由地理定位场景构成的精彩 AR 世界。作为玩家，你可以查看并参与其他用户创建的 AR 场景，身临其境地投入其中。在不同地点实时体验互动故事和冒险旅程。拍摄精彩瞬间，并通过照片或视频与他人分享难忘体验。此外，注册账号后，你还可以保存进度和互动记录，随时从上次离开的地方继续你的旅程。准备好开启一段充满探索、互动与临场感的 AR 冒险吧。

## 创作 + 分享（创作者模式）

在此模式下，你可以拥有并管理专属的 AR 体验和数字资产私人库。深入探索无限可能，在世界中构建世界，打造独一无二的沉浸式环境。通过设计故事和创造互动体验，将你的想象力转化为现实，吸引你的受众。为你的 AR 世界添加互动性和游戏化元素，让用户体验更加刺激有趣。完成创作后，轻松将其分享给你的受众，让他们踏上属于自己的精彩旅程。无论你是艺术家、故事创作者还是游戏设计师，创作者模式都为你提供展示才华、与世界分享创意作品的工具和平台。
