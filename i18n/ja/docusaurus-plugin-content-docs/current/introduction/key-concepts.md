# 主要コンセプト

## ストーリーテリングとしてのワールドビルディング

私たちは、ナラティブを構築し世界を創造することで、コミュニティがより緊密につながれると信じています。ELYSIUM では、AR を用いたストーリーを通じて、ロケーションベースの絆を生み出し、共有し、深める可能性が無限に広がっています。

## 誰のためのアプリ？

ELYSIUM は、ジオロケーション AR の可能性に興味を持つすべての人のためのアプリです。

AR で作品を展示・発信したいアーティスト、ローカル・グローバルに商品のマーケティング手法を拡げたい企業、学生の新しい学び方を模索する教育機関、あるいは単純に遊びを楽しみたい人まで、ELYSIUM はすべての人に何かを提供します。

## プレイヤー

プレイヤーは、クリエイターが構築したシーンやワールドを探索し、インタラクションできます。プレイヤーモードでは、アプリ内マップを使って、あらゆる場所でクリエイターが作成した AR シーンを見つけることができます。探索したいワールドやシーンを見つけたら、その場所をタップして ELYSIUM で周囲の世界を体験しましょう。

## クリエイター

クリエイターは、ライブラリ内のアセットを使って、ジオロケーションを指定した任意のエリアにワールドやシーンを構築できます。自分でバーチャルアセットをアップロードしてシーンに配置することも可能です。シーンはユーザーの現在地に基づいて表示され、クリエイターがロケーティブピンポイントで指定した範囲内で構築できます。シーンが公開されると、プレイヤーとクリエイターの両方が閲覧・インタラクションできるようになります。

## クリエイターネットワーク

クリエイター同士が自由に知識を求め・共有できる環境を目指しています。ELYSIUM のクリエイターコミュニティ Discord チャンネルを通じて、クリエイターはコミュニティを形成し、AR への共通の情熱を中心にアプリの最良の使い方を互いに学べます。Discord チャンネルには[こちら](https://discord.com/invite/gYMKNYHJRJ)から参加できます。
