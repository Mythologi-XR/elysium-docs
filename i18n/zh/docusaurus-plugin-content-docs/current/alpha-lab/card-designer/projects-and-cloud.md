# 项目与云端保存

拥有 [ELYSIUM 账户](./your-account)后，你可以将设计作品保存到云端，并通过右侧边栏的**项目**面板进行管理。

## 保存到云端

在侧边栏操作区域（ELYSIUM 标志按钮）点击 **"保存到 ELYSIUM"**。如果尚未登录，将首先打开登录界面。

![侧边栏操作区域中高亮显示的"保存到 ELYSIUM"按钮](/img/card-designer/53.png)

### 保存表单

填写云端保存的详细信息：

| 字段 | 是否必填 | 用途 |
|------|----------|------|
| **项目名称** | 是 | 从当前项目名称预填充 |
| **卡片名称** | 是 | 从卡片标题预填充 |
| **描述** | 是 | 设计的简短描述 |
| **状态** | 是 | **草稿**（进行中）或**已发布**（可分享） |

点击**保存**开始上传。进度条会显示上传百分比和文件大小。

![保存到 ELYSIUM 的弹窗，包含项目名称、卡片名称、描述和状态字段](/img/card-designer/54.png)

![上传进度条，显示完成百分比和 KB 文件大小](/img/card-designer/55.png)

### 保存后

保存完成后，你将看到四个选项：

| 操作 | 功能 |
|------|------|
| **继续编辑** | 关闭对话框，返回你的设计 |
| **切换项目** | 加载另一个云端 eCard |
| **分享卡片** | 为刚保存的卡片创建分享链接 |
| **创建新项目** | 通过快速设置向导重新开始 |

![保存后界面，显示继续编辑、切换项目、分享卡片和创建新项目操作按钮](/img/card-designer/56.png)

## 项目面板

右侧边栏中的**项目**面板（文件夹图标）显示你所有云端保存的 eCard。

### 登录前

如果你尚未登录，面板会显示**登录**和**创建账户**按钮供你开始使用。

![登录前的项目面板，显示登录和创建账户按钮](/img/card-designer/57.png)

### 你的 eCard 列表

登录后，你将看到已保存 eCard 的可滚动列表。每行显示：

- **项目名称**和**卡片名称**
- **状态标签** — 草稿（黄色）、已发布（绿色）或已同步（蓝色）
- 上次编辑的**时间**（例如"2小时前"、"3天前"）
- **展开箭头** — 点击查看描述和**打开**按钮

![项目面板，显示带有状态标签、时间戳的 eCard 列表，当前 eCard 以绿色高亮显示](/img/card-designer/58.png)

![展开的 eCard 行，显示描述文字和打开按钮](/img/card-designer/59.png)

当前正在编辑的 eCard 以绿色边框和复选标记高亮显示。

### 搜索与浏览

- 使用顶部的**搜索栏**即时筛选你的 eCard
- **上一页/下一页**按钮可浏览大型合集

![搜索栏在项目面板中筛选 eCard，显示分页控件](/img/card-designer/60.png)

### 加载云端 eCard

点击列表中的 eCard 将其加载到工作区。加载时会显示带有百分比和文件大小的下载进度条。

![下载进度条，显示从云端加载 eCard 时的百分比和文件大小](/img/card-designer/61.png)

:::caution
加载不同的 eCard 会替换当前工作区。如果有未保存的更改，确认对话框将询问你是要继续还是放弃更改。
:::

![确认对话框，显示"您有未保存的更改"，提供留下和离开按钮](/img/card-designer/62.png)

---

## 未保存更改保护

Card Designer 通过多种方式保护你的工作：

- **切换 eCard** — 如果在有未保存工作的情况下尝试加载不同的 eCard，将弹出确认对话框
- **退出登录** — 在有未保存更改的情况下登出前会发出警告
- **关闭浏览器** — 如果在有未保存工作的情况下尝试关闭标签页，将弹出原生"离开网站？"提示

![导航守卫弹窗，带有琥珀色警告、留下按钮（紫色）和离开按钮（灰色）](/img/card-designer/63.png)

:::tip
频繁使用"保存到 ELYSIUM"以保护你的工作。项目面板显示每张 eCard 的上次保存时间。
:::
