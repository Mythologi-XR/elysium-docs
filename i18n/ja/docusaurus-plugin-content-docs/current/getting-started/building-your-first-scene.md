# 初めてのシーン作成

## 初めてのワールドを作る

初めてのワールドを作るには、まずライブラリに必要なアセットが用意されていることを確認してください。準備ができたら、「マイワールド」ページまで左端にスワイプします。画面上部の「＋ 新規作成」ボタンをタップしてください。次に、「カバー画像を選択」をタップして写真ライブラリから希望の画像を選び、ワールドのカバー画像を設定します。ワールドにタイトルを付け、必要であれば説明文も追加できます（省略可）。最後に「ワールドを作成」をタップすれば完了です！これで初めてのワールドが作成され、いつでも構築を始められます。

## 初めてのシーンを作る

利用可能なワールドから任意のものを選択します。選択後、「シーンを編集・表示」ボタンをタップし、「新しいシーンを作成」または「＋」ボタンを選択します。現在地を示すピンが表示されたグローバルマップが開きます。シーンのジオフェンスを設定するには、シーンにしたいエリアの周囲の 4 点をタップして四角形を描きます。（AR に入るには、シーンの範囲内にいる必要があることに注意してください。）間違えた場合はやり直すことができます。完了したら「シーンを作成」ボタンをタップして進めます。「タイトルなし」と書かれた緑色のボタンが表示されます。それをタップして AR モードに入り、シーンの制作を開始しましょう。
