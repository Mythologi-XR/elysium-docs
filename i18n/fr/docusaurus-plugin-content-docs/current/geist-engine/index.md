# Moteur GEIST

**Moteur d'interaction et conception d'expériences**

Le moteur GEIST est le système de conception d'interactions au cœur d'ELYSIUM. Il propulse les relations entre les personnes, les lieux, les objets numériques et l'activité au sein des expériences RA.

Cette section est en cours de développement. Revenez pour une documentation détaillée sur :

- Primitives d'interaction et déclencheurs
- Logique spatiale et événements de proximité
- Comportements d'objets et machines à états
- Types de connexions (personnes, lieux, objets, activité)
- Conception d'expériences interactives engageantes
- Connexions entre réalités (RA, IoT, Web)
