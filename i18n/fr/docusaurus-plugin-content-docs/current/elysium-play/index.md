# ELYSIUM PLAY

**Aperçu et conception pour le jeu multiplateforme**

ELYSIUM PLAY est la couche d'expérience multiplateforme qui permet aux utilisateurs de découvrir, d'entrer et d'interagir avec des mondes RA sur mobile, web et interfaces portables.
