# 分享与隐私

Card Designer 提供两种分享设计的方式，并为访问权限提供完整控制。

## 两种分享方式

| | 本地预览 | 云端分享 |
|---|---|---|
| **方式** | 导出面板中的分享按钮 | 右侧边栏中的隐私与分享面板 |
| **跨设备有效** | 否（仅同一浏览器） | 是（任何拥有链接的人） |
| **密码保护** | 否 | 可选 |
| **有效期** | 永不过期 | 7、30 或 90 天（或永不过期） |
| **认领与协作** | 否 | 可选 |

**本地预览**适合在自己设备上快速查看。**云端分享**是向他人发送链接的最佳选择。

---

## 隐私控制

右侧边栏中的**隐私与分享**面板（盾牌图标）让你控制 eCard 的访问方式。你需要以已[登录](./your-account)状态使用已云端保存的 eCard 才能使用这些控制项。

![右侧边栏中的隐私与分享面板，显示四个隐私切换按钮——私密、启用分享、允许复制、允许认领](/img/card-designer/64.png)

四个切换按钮控制访问权限：

| 切换按钮 | 默认 | 功能 |
|----------|------|------|
| **私密** | 关闭 | 将你的 eCard 隐藏，不公开展示 |
| **启用分享** | 开启 | 允许你创建分享链接 |
| **允许复制** | 关闭 | 让接收者复制你的 eCard 并编辑他们自己的副本 |
| **允许认领** | 关闭 | 让接收者获取你的 eCard 的完整所有权 |

更改会自动保存——你将立即看到切换按钮更新。

## 创建云端分享链接

要与他人分享你的 eCard：

1. 确保你的 eCard 已[保存到云端](./projects-and-cloud)
2. 打开**隐私与分享**面板
3. 确保**启用分享**已开启
4. 选择你的分享链接设置：

### 密码保护

| 选项 | 说明 |
|------|------|
| **自动生成** | 系统为你创建一个安全的 9 位密码 |
| **自定义** | 设置你自己的密码（最少 9 个字符） |
| **无** | 无密码——任何有链接的人都可以查看 |

### 有效期

| 选项 | 到期时间 |
|------|----------|
| **7 天** | 创建后一周 |
| **30 天** | 创建后一个月 |
| **90 天** | 创建后三个月 |
| **永不** | 链接永久有效 |

5. 点击**创建分享链接**

![分享链接创建表单，显示密码模式选择器（自动生成、自定义、无）和有效期下拉菜单（7天、30天、90天、永不）](/img/card-designer/65.png)

## 管理你的分享链接

你的有效分享链接将以可展开列表的形式显示在隐私与分享面板中。对于每个链接，你可以：

- **复制 URL** — 点击复制图标获取完整链接
- **查看有效期** — 倒计时显示剩余时间（例如"15天后过期"）
- **查看密码** — 如果你创建了链接或重置了密码，密码将显示以便分享
- **重置密码** — 生成新密码或设置自定义密码（将使旧密码失效）
- **撤销** — 永久停用该链接

![分享链接列表，显示一个展开的链接，带有可复制的 URL、有效期倒计时、密码显示、重置密码按钮和撤销按钮](/img/card-designer/66.png)

新创建的链接会自动展开，方便你复制 URL 和密码。

## 查看者所见的内容

当有人打开你的分享链接时，他们将看到一个全屏 3D 查看器，完整呈现你的卡片设计：

- **3D 卡片预览** — 完整材质、光照和特效
- **信息栏** — 显示卡片标题、卡片类型和你的创作者名称
- **摄像预设** — 全部 5 个预设可用（竖向、英雄、悬浮、戏剧、展示）
- **卡片翻转** — 查看者可以在正面和背面之间翻转
- **音频** — 如果你添加了音频，将自动播放
- **复制链接** — 查看者可以从工具栏复制分享 URL

![云端分享播放器，显示 3D 卡片预览，信息栏显示卡片标题、类型和创作者名称，以及浮动工具栏](/img/card-designer/67.png)

如果查看者的设备不支持 WebGL，将显示静态图片预览而非 3D 渲染。

![WebGL 回退方案，当 3D 渲染不可用时显示卡片的静态图片预览](/img/card-designer/68.png)

:::note
分享播放器为只读模式——查看者无法修改你的设计，只能查看、翻转和与 3D 预览互动。
:::

## 密码保护分享

对于受密码保护的链接，查看者在访问预览前需要输入密码。

![密码验证表单，带有密码输入框和验证按钮](/img/card-designer/69.png)

**频率限制保护你的卡片安全：**
- 查看者有 **5 次机会**输入正确密码
- 5 次失败后，访问将被临时锁定，并显示可见倒计时
- 被锁定时会显示"请联系创作者获取帮助"的提示信息

![频率限制锁定界面，显示多次失败密码尝试后的倒计时和"联系创作者"提示信息](/img/card-designer/70.png)

## 认领与协作

如果你在隐私设置中启用了**允许认领**或**允许复制**，查看者可以做的不只是查看：

### 认领所有权

当**允许认领**开启时，查看者会看到一个**认领**按钮。认领操作将：

- 将 eCard 的完整所有权转移给查看者
- 为分享链接生成新密码（出于安全考虑）
- 原所有者失去编辑权限

![认领确认弹窗，黄色警告风格，说明所有权将转移给你](/img/card-designer/71.png)

:::caution
认领是单向转移。一旦有人认领你的 eCard，他们将成为新所有者，你将无法再对其进行编辑。
:::

### 编辑访问权（协作）

当**允许复制**开启时，查看者会看到一个**查看**按钮，允许他们申请编辑访问权。这将：

- 将他们添加为你 eCard 的**协作者**
- 让他们选择**立即编辑**（在 Card Designer 中打开）或**稍后编辑**
- 你保留所有者身份，拥有完整控制权

![编辑访问确认弹窗，紫色风格，显示立即编辑和稍后编辑选项](/img/card-designer/72.png)

:::note
查看者必须拥有 [ELYSIUM 账户](./your-account)才能认领或申请编辑权限。如果他们尚未登录，将自动弹出登录界面——登录后将自动继续之前的操作。
:::

## 角色说明

分享系统识别三种角色：

| 角色 | 身份 | 所见内容 | 可执行操作 |
|------|------|----------|----------|
| **所有者** | 你（创作者） | "所有者"标签 | 完整控制——编辑、管理分享、撤销链接 |
| **协作者** | 你赋予编辑权限的人 | "协作者"标签 | 在 Card Designer 中打开并编辑 eCard |
| **查看者** | 其他所有人 | 无标签 | 查看预览，认领或申请编辑权限（如已启用） |

![分享播放器，显示创作者名称旁边的所有者标签](/img/card-designer/73.png)

## 本地预览分享

导出面板和全屏工具栏中的**分享**按钮会创建快速本地预览链接。这会将你的卡片存储在浏览器本地存储中，并生成一个即时预览链接。

- 点击导出面板或全屏工具栏中的**分享**按钮
- 使用复制图标复制链接（绿色复选标记确认已复制）

:::caution
本地预览链接只在创建它们的**同一浏览器和设备**上有效。清除浏览器数据将永久删除它们。如需与他人分享，请使用隐私与分享面板中的云端分享功能。
:::
