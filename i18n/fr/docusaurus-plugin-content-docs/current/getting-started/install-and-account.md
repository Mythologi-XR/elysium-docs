# Installation et création de compte

## Obtenir l'application

Elysium est actuellement disponible uniquement sur iOS. Vous pouvez trouver l'application en la recherchant dans l'App Store. Elle est classée dans la catégorie Graphismes et Design. Il vous suffit de télécharger l'application et de la laisser s'installer sur votre appareil iOS, qu'il s'agisse d'un iPhone ou d'un iPad.

Téléchargez Elysium pour iOS ici :

[![Télécharger Elysium sur l'App Store](/img/app-store-button.svg)](https://apps.apple.com/app/id1594239469)

## Créer un compte

Une fois l'application installée sur votre appareil, appuyez sur l'icône Elysium pour l'ouvrir. Vous devriez accéder à un écran d'inscription vous permettant de créer un compte. Appuyez sur le bouton d'inscription et remplissez les informations sur l'écran suivant. Vous avez la possibilité de choisir un nom d'utilisateur afin que les autres utilisateurs de l'application sachent qui vous êtes et quelles scènes vous avez créées. Il vous sera demandé de vous inscrire avec une adresse e-mail et de créer un mot de passe d'au moins 8 caractères. Une fois ces informations fournies, vous devriez recevoir un e-mail vous demandant de confirmer votre inscription. Une fois confirmé, votre compte sera configuré et vous pourrez commencer à explorer Elysium.
