# 技巧与故障排除

## 键盘快捷键

| 按键 | 功能 |
|------|------|
| **1 - 5** | 切换摄像预设 |
| **F** | 翻转卡片 |
| **Esc** | 退出全屏预览 / 打开退出对话框 |
| **双击** | 翻转卡片（替代方式） |

:::tip
在文本输入框中输入内容时，摄像预设的键盘快捷键会暂停，因此不会因为输入内容而意外切换视角。
:::

## 故障排除

### 图片无法显示
- 确保使用 JPEG、PNG 或 WebP 格式的文件
- 将图片直接拖放到标有标签的区域——松开前注意看紫色高亮效果
- 如果图片非常大，尝试在上传前调整其大小

### 导出图片模糊
- 将分辨率倍数提高到 3x 或 4x
- 使用更高分辨率的原始图片
- 为获得最清晰的打印效果，使用**卡片面**模式而非**3D 场景**模式

### 3D 预览无法加载
- 确保在浏览器设置中已启用硬件加速
- 尝试使用其他浏览器（Chrome 或 Edge 效果最佳）
- 更新你的显卡驱动程序

### 我的分享链接对他人无效

**对于本地预览链接**（通过导出面板的分享按钮创建）：这类链接只在创建它们的同一浏览器和设备上有效。请改用[云端分享](./sharing-and-privacy)，生成对任何人都有效的链接。

**对于云端分享链接**，请检查以下常见原因：
- **"已过期"** — 链接已超过有效期。请联系创作者生成新链接
- **"已撤销"** — 创作者已永久停用此链接
- **"分享已禁用"** — 创作者关闭了此 eCard 的分享功能
- **"尝试次数过多"** — 你已超过 5 次密码输入次数。请等待倒计时重置

### 无法登录或创建账户
- 仔细检查你的电子邮箱和密码——密码区分大小写
- 如果会话已过期，请重新登录（会话会自动刷新，但长时间不活动后可能超时）
- 创建账户时，确保密码至少 8 个字符，且邮箱未被使用

### 云端保存无法正常工作
- **上传似乎卡住了** — 检查网络连接后重试
- **"描述为必填项"** — 描述字段是必填的——保存前请填写
- **项目面板中看不到 eCard** — 尝试搜索，或确认你已登录正确的账户

### GLB 导出显示错误

![导出面板显示 GLB 验证失败后的红色错误状态，展开显示错误列表](/img/card-designer/45.png)

- 尝试在导出设置中开启**烘焙纹理**

![导出面板显示 GLB 导出后带有"验证通过，有警告"的琥珀色警告](/img/card-designer/46.png)

- 如果错误持续，重新加载页面，重新上传图片，然后再试
