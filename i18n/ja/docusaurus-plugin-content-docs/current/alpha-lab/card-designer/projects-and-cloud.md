# プロジェクトとクラウド保存

[ELYSIUM アカウント](./your-account)をお持ちの場合、デザインをクラウドに保存し、右サイドバーの **プロジェクト** パネルから管理できます。

## クラウドへの保存

サイドバーアクションエリアにある **「ELYSIUM に保存」**（ELYSIUM ロゴボタン）をクリックしてください。まだサインインしていない場合は、先にサインイン画面が開きます。

![サイドバーアクションエリアでハイライトされた「ELYSIUM に保存」ボタン](/img/card-designer/53.png)

### 保存フォーム

クラウド保存の詳細を入力してください：

| フィールド | 必須 | 用途 |
|----------|------|------|
| **プロジェクト名** | はい | 現在のプロジェクト名から自動入力されます |
| **カード名** | はい | カードのタイトルから自動入力されます |
| **説明** | はい | デザインの短い説明 |
| **ステータス** | はい | **下書き**（作業中）または **公開**（共有可能） |

**保存** をクリックしてアップロードを開始します。進行状況バーにアップロードの割合とファイルサイズが表示されます。

![プロジェクト名、カード名、説明、ステータスフィールドを含む「ELYSIUM に保存」モーダル](/img/card-designer/54.png)

![割合とファイルサイズ（KB）を表示するアップロード進行状況バー](/img/card-designer/55.png)

### 保存後

保存が完了すると、4 つのオプションが表示されます：

| アクション | 内容 |
|----------|------|
| **編集を続ける** | ダイアログを閉じてデザインに戻ります |
| **プロジェクトを切り替える** | 別のクラウド eCard を読み込みます |
| **カードを共有** | 保存したカードの共有リンクを作成します |
| **新しいプロジェクトを作成** | クイックセットアップウィザードで新しいプロジェクトを始めます |

![編集を続ける、プロジェクトを切り替える、カードを共有、新しいプロジェクトを作成のアクションボタンが表示された保存後画面](/img/card-designer/56.png)

## プロジェクトパネル

右サイドバーの **プロジェクト** パネル（フォルダアイコン）には、クラウドに保存したすべての eCard が表示されます。

### サインイン前

ログインしていない場合、パネルには **サインイン** と **アカウントを作成** のボタンが表示されます。

![サインインとアカウントを作成ボタンを表示したサインイン前のプロジェクトパネル](/img/card-designer/57.png)

### eCard リスト

サインイン後、保存済み eCard のスクロール可能なリストが表示されます。各行には以下が表示されます：

- **プロジェクト名** と **カード名**
- **ステータスバッジ** — 下書き（黄）、公開（緑）、同期済み（青）
- 最終編集からの **経過時間**（例：「2 時間前」、「3 日前」）
- **展開矢印** — クリックすると説明と **開く** ボタンが表示されます

![ステータスバッジ、タイムスタンプ、緑でハイライトされたアクティブな eCard を表示するプロジェクトパネルの eCard リスト](/img/card-designer/58.png)

![説明テキストと開くボタンを表示する展開された eCard 行](/img/card-designer/59.png)

現在作業中の eCard は緑のボーダーとチェックマークでハイライトされます。

### 検索と閲覧

- 上部の **検索バー** を使って eCard を即座にフィルタリングできます
- **次へ/前へ** ボタンで大きなコレクションをナビゲートできます

![ページネーションコントロールが表示されたプロジェクトパネルで eCard をフィルタリングする検索バー](/img/card-designer/60.png)

### クラウド eCard の読み込み

リスト内の eCard をクリックしてワークスペースに読み込みます。読み込み中は割合とファイルサイズが表示されるダウンロード進行状況バーが表示されます。

![クラウドから eCard を読み込む際の割合とファイルサイズを表示するダウンロード進行状況バー](/img/card-designer/61.png)

:::caution
別の eCard を読み込むと、現在のワークスペースが置き換えられます。未保存の変更がある場合、作業を続けるか変更を破棄するかを確認するダイアログが表示されます。
:::

![「未保存の変更があります」という確認ダイアログと、留まるボタン・移動するボタン](/img/card-designer/62.png)

## 未保存の変更の保護

Card Designer はいくつかの方法でデータを守ります：

- **eCard の切り替え** — 未保存の作業がある状態で別の eCard を読み込もうとすると確認ダイアログが表示されます
- **サインアウト** — 未保存の変更がある場合、ログアウト前に警告が表示されます
- **ブラウザを閉じる** — 未保存の作業がある状態でタブを閉じようとすると、ネイティブの「サイトを離れますか？」プロンプトが表示されます

![アンバー色の警告、留まるボタン（紫）、移動するボタン（グレー）が表示されたナビゲーションガードモーダル](/img/card-designer/63.png)

:::tip
「ELYSIUM に保存」を使って頻繁に保存し、作業を安全に保ちましょう。プロジェクトパネルには各 eCard の最終保存時刻が表示されます。
:::
