# インストールとアカウント設定

## アプリを入手する

ELYSIUM は現在 iOS のみで利用可能です。App Store で検索してアプリを見つけてください。「グラフィックス＆デザイン」カテゴリに登録されています。アプリをダウンロードし、iPhone または iPad にインストールしてください。

iOS 版 ELYSIUM をここからダウンロード：

[![App Store で ELYSIUM をダウンロード](/img/app-store-button.svg)](https://apps.apple.com/app/id1594239469)

## アカウントを作成する

アプリがデバイスにインストールされたら、ELYSIUM のアイコンをタップして開きます。アカウント登録画面が表示されます。サインアップボタンをタップし、次の画面に必要な情報を入力してください。他のユーザーがあなたを識別し、あなたが作成したシーンを確認できるよう、ユーザー名を設定するオプションがあります。メールアドレスでの登録と、8文字以上のパスワードの設定が必要です。情報を入力すると、登録確認のメールが届きます。確認が完了したら、アカウントの設定が完了し、ELYSIUM での探索を始められます。
